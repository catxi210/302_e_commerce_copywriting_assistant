## 开发&部署
1. 克隆项目 `git clone https://github.com/302ai/302_e_commerce_copywriting_assistant`
2. 安装依赖 `npm install`
3. 配置302的API KEY 参考.env.example
4. 运行项目 `npm dev`
5. 打包部署 `docker build -t coder-generator . && docker run -p 3000:3000 coder-generator`
